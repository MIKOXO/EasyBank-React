
# EasyBank Landing Page

This is a Clone project for the Landing page of EasyBank from FrontEnd Mentor Challenges.



## Tech Stack

**Client:** React, TailwindCSS




## Features

- Fully responsive
- HamBurger Menu


Go to the project directory

```bash
  cd my-project
```

```bash

```

